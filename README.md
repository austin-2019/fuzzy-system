### Getting Started

##### Go to NASA's site for Open APIs and get a key at https://api.nasa.gov/#apply-for-an-api-key

#### Prerequisites

- [Node.js](https://nodejs.org/en/download/)

#### Installation

```bash
git clone https://github.com/austin-2019/fuzzy-systems.git
cd fuzzy-systems
npm install
```

#### Usage

```bash
npm start
```
